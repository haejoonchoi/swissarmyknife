# Filesig

* [swissarmyknife](README.md)
* [Code](filesig)

An improved version of `sha1sum` that will recurse a directory structure

## Example

```
./filesig generate --format=full /path/to/files.txt .
./filesig verify /path/to/files.txt .
```
