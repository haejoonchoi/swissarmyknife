# Deduper

* [swissarmyknife](README.md)
* [Code](deduper)

This is a reasonably useful file deduplicator, based on and inspired by [`find_duplicates.py`][find-duplicates].

## Example

```
./deduper --strategy keep-first --dry-run --verbose .
```
